<Note>
Generating an estimate from property photos is no longer available. Estimates are now built with the step-by-step **Calculator** wizard.
</Note>

To create an estimate, open **Estimates**, choose **New Estimate**, and work through the Calculator — pick the client and work areas, add items, and review the breakdown.

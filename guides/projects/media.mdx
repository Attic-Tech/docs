The **Media** tab is where every photo, video, and document for a project lives. Crews capture what they see on site, office staff add documents, and everyone can browse it all in one gallery — organized into folders so the right files are easy to find.

<Note>
Media is shared across the whole project. Anything you upload here is available to anyone working the job.
</Note>

## Open the gallery

<Steps>
  <Step title="Open the project">
    Open the project from the client, then click the **Media** tab.
  </Step>
  <Step title="Browse the gallery">
    You'll see the media grid, with **Folders** down the side. The client's name and address sit at the top so you always know whose job you're looking at.
  </Step>
</Steps>


## Folders

The **Folders** list on the side groups media by what it's for, with a running count next to each one:

- **All Media** — everything on the project.
- **Inspection**, **Existing Damage**, **HVAC**, **Roofing**, **Setup**, **Completion**, **Documents**, **General**, and **Client Collection**.

Click a folder to show only its files. Click **All Media** to see everything again.

<Tip>
The folder you're viewing decides where new uploads land. Open the folder you want **before** you upload, and your files go straight into it. New files default to **General** when no folder is selected.
</Tip>

## Add media

You can add files two ways — both work on phones and computers.

<Steps>
  <Step title="Pick the right folder first">
    Click the folder you want the files to go into (or stay on **All Media** for General).
  </Step>
  <Step title="Add your files">
    Click **Upload Files**, or drag files onto the **Drag files here** drop zone and drop them. On a phone, tap the drop zone to choose photos or files.
  </Step>
  <Step title="Watch them upload">
    Each file shows its progress as it uploads, then settles into the grid when it's done. You can keep adding more while others finish.
  </Step>
</Steps>


<Note>
**What you can upload.** Images, videos, and documents are accepted, each up to its own size limit (shown right on the drop zone, for example "Images · Videos · Documents"). If a file is the wrong type or too large, you'll get an **Upload failed** message telling you which file and why — the rest still upload.
</Note>

## View a single file

Click any thumbnail to open it full size in the viewer. From there you can:

- Move between files with the **left and right arrows** — the viewer keeps loading more as you go.
- See your place with the **"X of Y"** counter at the bottom.
- Open the **menu** (the three-dot button) to **Download** or **Delete** the file.

Videos play right in the viewer, and documents (like PDFs) open inline so you can read them without leaving the page.


## Download a file

To save a copy, open the file and choose **Download** from its menu. You can also download straight from a thumbnail using its menu, without opening the viewer first.

## Delete a file

<Steps>
  <Step title="Choose Delete">
    From a thumbnail's menu or the viewer's menu, choose **Delete**.
  </Step>
  <Step title="Confirm">
    You'll be asked **Delete media?** to confirm. Click **Delete** to remove it, or **Cancel** to keep it.
  </Step>
</Steps>

<Note>
Deleting media is permanent and can't be undone, and it's a more protected action — only **management roles** can delete project files. If you don't see a **Delete** option, your role doesn't include it.
</Note>

## Tips for tidy media

<Tip>
Drop files into the folder that matches their purpose — **Existing Damage** before work starts, **Completion** when you finish — so anyone reviewing the job can find what they need at a glance.
</Tip>

Project media works hand in hand with the rest of the project. For the bigger picture of how everything connects, see [Work a project](/guides/projects/work-a-project).
